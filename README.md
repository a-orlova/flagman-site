# ФЛАГМАНЪ — Речная флотилия Москвы

Продающий лендинг для бизнеса по речным прогулкам по Москве‑реке. Бренд построен вокруг флагманского теплохода **«Флагманъ»**; второй корабль — **«Лоцманъ»** — представлен как камерная альтернатива.

## Запуск

Это статический сайт — собирать ничего не нужно.

**Локально:**

1. Откройте `index.html` двойным кликом в браузере *или*
2. Запустите простой локальный сервер (рекомендуется, чтобы корректно открылись PDF‑меню):

```bash
# Python 3
python -m http.server 8000

# или, если есть Node
npx serve .
```

Откройте `http://localhost:8000`.

**На хостинге:** залейте все файлы и папки (`index.html`, `styles.css`, `script.js`, `images/`, `information/`) в корень любого статического хостинга — Beget, Reg.ru, Vercel, Netlify, GitHub Pages.

## Структура

```
index.html                — единственная страница (Hero, флот, маршрут, меню, галерея, форма)
styles.css                — стили (A11Y‑палитра, адаптив, анимации)
script.js                 — интерактив (анимация кораблика по маршруту, лайтбокс, форма, PDF‑viewer)
images/anchor-icon.svg    — иконка якоря (фавикон, навигация)
images/флагман/           — фотографии Флагмана (используются в галерее, hero, карточке)
images/лоцман/            — фотографии Лоцмана
information/              — PDF‑меню (открываются прямо на сайте через iframe)
```

## Кастомизация

### Цвета

Все цвета в одном месте — `:root` в `styles.css`:

```css
--c-primary:  #2C5A78;   /* основной синий */
--c-gold:     #8A6F28;   /* золотой текст */
--c-gold-2:   #B89D5C;   /* декоративное золото */
--c-bg:       #F4F8FB;   /* фон */
--c-bg-2:     #FBF7EE;   /* кремовый фон секций */
```

Палитра проверена на контрастность WCAG AA: основной текст 14:1, акценты 5–6.5:1.

### Подключение настоящего платёжного шлюза

В `script.js` найдите блок:

```js
// === ИНТЕГРАЦИЯ С ПЛАТЁЖНЫМ ШЛЮЗОМ ===
setTimeout(() => {
  alert('Демо-режим: ...');
  ...
}, 900);
```

Замените `setTimeout(...)` на отправку формы на свой бекенд / редирект:

```js
const data = new FormData(form);
fetch('/api/create-order', { method: 'POST', body: data })
  .then(r => r.json())
  .then(({ payUrl }) => { window.location.href = payUrl; });
```

Популярные шлюзы:
- **ЮKassa** (Сбербанк) — [yookassa.ru](https://yookassa.ru)
- **CloudPayments** — [cloudpayments.ru](https://cloudpayments.ru)
- **Тинькофф Касса** — [tinkoff.ru/kassa](https://www.tinkoff.ru/kassa/)

### Меню (PDF)

Файлы лежат в `information/`. Если меню сменилось — просто положите новый PDF с тем же именем, либо поменяйте `data-pdf="..."` в `index.html` на табах:

```html
<button class="menu__tab" data-pdf="information/новое-меню.pdf">...</button>
```

### Маршрут (анимация кораблика)

8 точек заданы в SVG в секции `route__stage` в `index.html`. Чтобы изменить достопримечательности — отредактируйте список `route__list` ниже SVG.

### Расписание

Сейчас 4 рейса захардкожены в `<select id="bookTime">` в `index.html`. Если нужно подгружать из CSV/Excel/API — это правится в `script.js` (можно создать функцию, которая при выборе даты подгружает доступные слоты).

## Адаптивность

Лендинг построен mobile‑first. Брейкпоинты:

- < 480 px — мобильные
- 480 – 768 px — планшеты в портрете
- 768 – 980 px — планшеты, маленькие ноутбуки
- ≥ 980 px — десктоп

Меню сворачивается в бургер на экранах < 920 px. Галерея и сетки переключаются на 2 / 3 / 4 колонки.

## Доступность (A11Y)

- Семантические теги (`header`, `nav`, `section`, `article`, `aside`, `figure`)
- `aria-*` для модалок, лайтбокса, табов и навигации
- `prefers-reduced-motion` — отключает анимации для пользователей, которые этого попросили в системных настройках
- Фокус‑видимость для клавиатуры (`:focus-visible`)
- Контрастность по WCAG AA
